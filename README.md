## Task Management

This repository is a task manager built using Laravel and Livewire.
It is a simple app for an user to create, change and delete tasks from projects that they can create.

## How to launch

Clone the repository

Change .env file to reflect your database connection and other configuration

Run "composer install" for the necessary packages

Run "php artisan migrate" for the necessary migrations

## How to use

Access "/" path

Register an user on the top right navbar

After logging in, create your first project.

Then, you can create tasks, and drag and drop then to change the priority between them.
